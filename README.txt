HOW TO OPEN THIS PORTFOLIO LOCALLY

1. Create a folder on your Desktop named: My Portfolio
2. Put these files inside it:
   - index.html
   - enhanced_image.jpg
3. Open the folder in VS Code.
4. Right-click index.html and choose "Open with Live Server".
5. If you do not have Live Server, double-click index.html to open it in Chrome.

Important:
- Keep enhanced_image.jpg in the SAME folder as index.html.
- Do not rename enhanced_image.jpg unless you also change the image src inside index.html.
- To edit text, open index.html in VS Code and search for the text you want to change.

For GitHub Pages later:
- Your main file must be named exactly index.html.
